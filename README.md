# Sombra
Sombra是一款基于C++的现代化、面向Linux的、MVVM模式的Web开发框架

## 如何运行
运行需要的环境：  
* Linux (Windows 下不支持)  

运行前需要安装：

* bower
* npm
* git
* g++

运行前需要准备：  

    bower install
    npm install

启动项目所需的命令：

    ./start.sh

之后启动浏览器，访问下面地址：

    http://localhost:9000/

如果能够正常体验应用，那么恭喜，你已经完成了该项目的编译、链接和运行！
